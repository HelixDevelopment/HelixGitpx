# HelixGitpx

HelixGitpx is a federated Git proxy. One namespace across many Git hosts — no
vendor lock-in, no lost state when a provider has an outage.

This site is the public documentation home. Start with
[Install](./getting-started/install) then try the
[first-repo walkthrough](./getting-started/first-repo).
