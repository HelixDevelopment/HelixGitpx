# M8 GA Implementation Plan

**Goal:** Take HelixGitpx to General Availability: multi-region, performance-validated, DR-drilled, security-hardened, fully covered, and with GA artifacts shipped.

**Architecture:** Extends existing patterns. Two-region active-passive via Kustomize overlays + MirrorMaker 2 + CNPG logical replication. Perf, chaos, and coverage are CI workflows + tooling under `tools/`. GA adds docs site, trust center route, billing-service, and release artifacts.

**Tech Stack:** k6, Litmus, MirrorMaker 2, CNPG, CoreDNS+geoip, Docusaurus, Stripe.
