# Script — 19 Org management, RBAC, residency

**Track:** Admins · **Length:** 8 min · **Goal:** admin confidently manages their org on day one.

## Body

1. **Create an org** — 0:30 – 1:30.
   Name, slug, residency (EU/UK/US).
2. **Invite members** — 1:30 – 3:00.
   Email invite, role selection.
3. **Roles + teams** — 3:00 – 4:30.
   Owner / admin / member; team = collection with a role.
4. **Repository RBAC** — 4:30 – 5:30.
   Per-repo roles overlay org roles.
5. **SSO (Scale+)** — 5:30 – 6:30.
   SAML config, just-in-time provisioning.
6. **Data residency switch** — 6:30 – 7:30.
   Flip via `SetOrgResidency`; expected migration window.

## Wrap-up (7:30 – 8:00)
"Owners own tenancy. Admins own policy. Members do the work."
