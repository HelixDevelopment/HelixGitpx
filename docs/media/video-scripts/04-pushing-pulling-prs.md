# Script — 04 Pushing, pulling, PRs

**Track:** Getting started · **Length:** 6 min · **Goal:** viewer can open, review, merge a PR across federated hosts.

## Body

1. **Create a branch, push** — 0:30 – 1:30.
   `git checkout -b feat/readme; … ; git push`.
2. **Open a PR in HelixGitpx web** — 1:30 – 2:30.
   Title, description, reviewer selection.
3. **Verify fanout** — 2:30 – 3:30.
   Same PR visible on GitHub + GitLab with identical metadata.
4. **Review and comment** — 3:30 – 4:30.
   Inline comment from HelixGitpx → mirrored to GitHub.
5. **Merge and watch mirrors** — 4:30 – 5:30.
   Squash merge; branches deleted everywhere; audit log captures all.

## Wrap-up (5:30 – 6:00)
"Your workflow didn't change. The blast radius did."
