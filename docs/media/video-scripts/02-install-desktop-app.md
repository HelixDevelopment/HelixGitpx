# Script — 02 Install the desktop app

**Track:** Getting started · **Length:** 2 min · **Goal:** viewer has the desktop app running.

## Cold open (0:00 – 0:10)
Smash cut: web app side-by-side with desktop app. "Same everything, no browser."

## Body

1. **Pick your OS** — 0:10 – 0:25.
   Show the downloads page: `.dmg`, `.msi`, `.AppImage`, `.deb`, `.rpm`.
2. **Install — macOS** — 0:25 – 0:45.
   Drag to Applications, open, accept Gatekeeper prompt.
3. **Install — Windows** — 0:45 – 1:05.
   MSI prompt; start menu entry; first-run login.
4. **Install — Linux (AppImage)** — 1:05 – 1:25.
   `chmod +x`; `./HelixGitpx-*.AppImage`.
5. **First run** — 1:25 – 1:50.
   OIDC browser handoff; desktop app reports "signed in".

## Wrap-up (1:50 – 2:00)
"Next: bind your first repo." End card.
