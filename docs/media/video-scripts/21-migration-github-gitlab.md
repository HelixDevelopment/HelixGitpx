# Script — 21 Migrating from GitHub / GitLab

**Track:** Admins · **Length:** 12 min · **Goal:** team lead walks through a real migration with no downtime.

## Body

1. **Philosophy: federate, don't cut over** — 0:30 – 1:30.
   Add HelixGitpx as an additional host, not a replacement.
2. **Phase 1 — bind existing repos** — 1:30 – 3:30.
   Bulk-import script; scope a 5-repo pilot.
3. **Phase 2 — mirror-verify** — 3:30 – 5:30.
   1–2 weeks of dual-writes; parity checks.
4. **Phase 3 — flip primary** — 5:30 – 8:00.
   Update developer docs; rotate PAT hosts.
5. **Phase 4 — graduate or keep as insurance** — 8:00 – 9:30.
   Choose: remove old host, or keep as a mirror.
6. **Artefact coverage matrix** — 9:30 – 11:00.
   What migrates (history, PRs, issues, comments, labels) and what doesn't.

## Wrap-up (11:00 – 12:00)
"You never have to be 'done'. That's the point."
